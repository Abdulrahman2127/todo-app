# 📝 Todo App

A simple and modern task management application built with React and Material UI.

## ✨ Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Filter tasks (All / Completed / Not Completed)
- Save tasks using Local Storage

## 🛠️ Built With

- React
- Material UI (MUI)
- JavaScript
- CSS
- Local Storage

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/Abdulrahman2127/todo-app.git
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm start
```
